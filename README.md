# ThEbookApp (2021)
An e-book reader app as a learning media between lecturers and students of FIK UPNVJ 

- The application was built using Android Studio version 4.2.1 with Java as the programming language.
- Using Firebase as a database and storage to store ebooks and user profile photos in the ThEBOOK application.
- Using Gradle version 6.7.1 and SDK version 30
- Using various libraries such as pdf viewer, file loader, image picker, imager downloader, bottom navigation.
